# Code Review Skill

Perform thorough code reviews with focus on best practices, security, and performance.

## Usage

Use this skill when reviewing code changes, pull requests, or conducting code audits.

## Guidelines

- Check for security vulnerabilities (SQL injection, XSS, CSRF)
- Verify proper error handling and input validation
- Ensure TypeScript type safety
- Review for performance optimizations
- Check code style consistency
- Verify test coverage for new features
